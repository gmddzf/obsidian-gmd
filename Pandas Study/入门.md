# 导入pandas库、读取CSV文件
```python
import pandas as pd
#假设movies.csv在data目录下
movies_df=pd.read_csv('date/movies_sample.csv')
```
>[!note]- 代码执行说明
>**第一行**：`import pandas as pd`
> -  导入pandas库，并起一个简称`pd`
> -  行业惯例，所有人都这么写
>**第四行**：`movies_df = pd.read_csv(...)`
>- 调用`pd.read_csv()`函数
>-  将结果赋值给变量`movies_df`
>- 文件名用引号括起来（字符串
